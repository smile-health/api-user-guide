## About
API Integration Service functions to integrate data between SMILE web and mobile applications and 
other systems, both as a source and destination for data exchange. This service acts as a backend 
connector to optimize data management, improve tracking accuracy, and increase overall business 
operational efficiency. The integration service in SMILE is carried out to record order integration,
post picking order, and disposal management.

---

## Associated Components
To support these functionalities, the SMILE system is integrated with several essential components, including:
1. Order: Record order integration and post picking order;
2. Disposal: Check disposal instruction data status from WMS.
